# Rest API

Esta API implementa a especificação [JSON:API](https://jsonapi.org/). Documentação dos endpoints se encontra no arquivo *postman_collection.json*.

## Download

```
git clone https://github.com/joaoeduardo/teste.git
```

## Instalação

```
composer install

php artisan migrate
```

## Subir servidor
```
php artisan serve
```

## Testes
```
./vendor/bin/phpunit
```
